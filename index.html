How to use the Fishbone Diagram Tool

Edit text

  • Click any category label, heading, or bullet point to edit.

Select and add

  • Click a heading to select it.
  • Use the floating + Heading / + Bullet buttons next to the selected heading.

Move headings

  • Drag using the ⠿ handle. Headings remain attached to the category bone.

Problem / effect

  • Click the arrow text to edit.
  • Drag ↕ to move it.
  • Resize the box using the bottom-right resize handle.

Save and export

  • Export JSON saves; Import JSON reloads.
  • Export PNG/PDF captures the diagram for reporting.
