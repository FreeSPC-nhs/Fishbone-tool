Fishbone Diagram Tool – help & guidance

This tool lets you build Ishikawa / fishbone diagrams in the browser. Data stays on your device.

Typical workflow

  • Enter the problem / effect on the left (“Medication errors on ward X”).
  • Add categories such as People, Process, Equipment, Environment, Materials and Measurement.
  • Add causes within each category, and optionally select a parent cause to create sub-causes.
  • Use the PNG or PDF export buttons in the header to capture the diagram for reports or slides.
  • Export the model as JSON if you want to reload and edit it later.

Tips

  • Click any category box in the diagram to edit that category.
  • Click a cause in the table to edit it in the form.
  • Appearance settings (box height, spacing, font size) can be tuned to fit your document layout.
  • CSV export provides a simple table of categories and causes, suitable for spreadsheets.
