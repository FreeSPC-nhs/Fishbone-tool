How to use the Fishbone Diagram Tool

Edit text

  • Click any category label, heading, or bullet point to edit.
  • Click outside the text when finished.

Select a heading

  • Click a heading block to select it (it highlights).
  • “+ Bullet” in that category adds a bullet to the selected heading.
  • If nothing is selected, “+ Bullet” adds to the last heading in that category.

Move headings

  • Use the ⠿ grab handle to drag a heading up/down.
  • The rib stays attached to the diagonal category bone as you move.

Problem / effect box

  • Edit the text in the arrow by clicking it.
  • Move the text using the small ↕ handle.

Save, reload and export

  • Export JSON to save your work and reload later with Import JSON.
  • Export PNG/PDF to capture the diagram for reports and slides.

Settings

  • Use Settings to adjust colours, font size, and bone/rib formatting.
